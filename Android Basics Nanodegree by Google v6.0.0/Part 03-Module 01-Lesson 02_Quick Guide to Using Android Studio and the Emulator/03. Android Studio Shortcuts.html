03. Android Studio Shortcuts

Action Explanation
Cmd+Shift + A (Ctrl + Shift + A Windows or Linux PC) Search through all of Android Studio

(In my opinion, the most important shortcut!)
Code > Reformat Code To ensure correct indentation and the right order of attributes
Connect device (Emulator or Physical) > Android Monitor > Screen Capture Obtain a screenshot from the device (Additional Info)
Android Monitor > Screen Record Android Monitor lets you record an MP4 video from your hardware device for a maximum of three minutes. You can use the video for your marketing materials as well as for debugging, for example (Additional Info)
View > Enter Presentation Mode Presentation Mode - Useful when presenting something something from the IDE to a group so everyone can see it.

It will increases the font size and hides the toolbars.
Shift+Cmd+V Instead of pulling up just the most recent copy-paste (Cmd+V) Android Studio remembers all your copy-pastes in the Clipboard Buffer.

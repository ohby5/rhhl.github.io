01. Quick Guide to Using Android Studio and the Emulator

Tour Intro

In order to be an effective Android developer, you’ll need to be familiar with the tool that you’ll be using more than 90% of the development time - Android Studio!

You’re not alone if the first time you opened Android Studio and recoiled from the aftershock after seeing all the features.

The sheer number of buttons, settings, and configurations in Android Studio can definitely cause option paralysis, stopping you from even starting.

Don’t fret. In this lesson we will go over the main components of the IDE (Integrated Development Environment):

  • The Tool Windows
  • Android Studio Shortcuts
  • Android Studio Updates and Understanding Release Notes
  • Image Compression

And before you know it you’ll be an old pro at finding your way around this helpful tool.

03. Introduction: Pokemon App

Introduction: Pokemon App

In this section, we will be applying what we learned to code.

We will be making a simple Pokemon app that lists out Pokemons with their respective attributes using the JSON at this link: https://raw.githubusercontent.com/Biuni/PokemonGO-Pokedex/master/pokedex.json

What is Pokemon?

Pokemon is a Japanese game where players collect small monsters called Pokemons who have specific traits such as their experience level, height, weight, ability, etc. They compete with each other in battles and each one has specific characteristics. For example, each Pokemon will have their own specific type of candy that they eat, which helps them evolve. When a Pokemon evolves it gains new abilities or improves its existing abilities. You can read more about the Pokemon game here: https://en.wikipedia.org/wiki/Gameplay_of_Pok%C3%A9mon

We will practice making a HTTP Request and parsing through the JSON nodes.

The skeleton code is provided on our Github at this link: https://github.com/udacity/Pokemon
